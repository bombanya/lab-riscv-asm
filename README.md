### Результаты работы

Был реализован вариант с возвратом номера строки из функции, обрабатывающей массив.

* **lab.S** - программа на RISC-V ASM
* **lab.c** - программа на C
* **lab.dump** - дамп-файл программы на C

Дамп был получен для исполняемого файла, скомпилированного с -O1.  
В таком варианте из примечательного можно отметить разве что то, что
массив значений хранится на стеке и в функцию для его обработки 
просто передается соответствующий адрес. Также в main функции
присутствует код, выполняющий сохранение/восстановление некоторых
из callee-saved регистров, которые используются в функции для вычислений.  
В остальном полученный код соответствует тому, что написал я сам.
